Computes the weighted average of a fuzzy set represented in an AccumulatedTerm as \(y = \dfrac{\sum_i w_iz_i}{\sum_i w_i} \), where \(w_i\) is the activation degree of term \(i\), and \(z_i = \mu_i(w_i) \).

If the accumulation operator in the given AccumulatedTerm is fl::null, then the WeightedDefuzzifier yields the typical operation defined by the given equation. Otherwise, the accumulation operator is utilized to compute a single weight from the weights of the repeated terms in the fuzzy set, and then proceed with the given equation.

Parameters
termis the fuzzy set represented as an AccumulatedTerm
minimumis the minimum value of the range (only used for Tsukamoto)
maximumis the maximum value of the range (only used for Tsukamoto)
