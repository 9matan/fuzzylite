Member Enumeration Documentation

enum fl::Engine::Type
Enumerator
Mamdani 

When the output variables have IntegralDefuzzifiers.

Larsen 

When Mamdani and AlgebraicProduct is the implication operator of the rule blocks.

TakagiSugeno 

When output variables have WeightedDefuzzifiers of type TakagiSugeno and the output variables have Constant, Linear, or Function terms.

Tsukamoto 

When output variables have WeightedDefuzzifiers of type Tsukamoto and the output variables only have monotonic terms (Concave, Ramp, Sigmoid, SShape, and ZShape)

InverseTsukamoto 

When output variables have WeightedDefuzzifiers of type TakagiSugeno and the output variables do not only have Constant, Linear or Function terms.

Hybrid 

When output variables have different defuzzifiers.

Unknown 

When output variables have no defuzzifiers.

Configures the engine with clones of the given operators.

The given operators are automatically deleted within this method.

Parameters
conjunctionis the operator to process the propositions joined by and in the antecedent of the rules
disjunctionis the operator to process the propositions joined by or in the antecedent of the rules
implicationis the operator to modify the consequents of the rules based on the activation degree of the antecedents of the rules
accumulationis the operator to accumulate the resulting implications of the rules
defuzzifieris the operator to transform the accumulated implications into a single scalar value
